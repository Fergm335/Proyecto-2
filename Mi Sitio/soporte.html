Hay dos requisitos para usar una aplicación web progresiva: un navegador compatible y un servicio habilitado. Webs como IsServicesWorrkedready permiten verificar la compatibilidad de los navegadores con PWA y otros como CanIUse, se especializan en enumerar la implementación de esta tecnología en cada por cada versión y navegador.

Desglosando el estado actual de soporte:

  • Navegador de escritorio (Soporte completo): Chrome, Firefox, Opera, Edge, Safari.
  • Navegador de escritorio (soporte parcial / versión desactualizada): QQ, Baidu.
    •
  • Navegador móvil (soporte completo): Chrome, Firefox, Safari, UC Browser, Samsung internet, Mint Browser, Wechat.
    •
  • Navegador móvil (soporte parcial / versión desactualizada): QQ, Android Browser, Opera Mobile.
